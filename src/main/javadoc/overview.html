Calculator Spring Boot Application

Esta aplicación web proporciona una calculadora completa con operaciones matemáticas básicas y avanzadas, desarrollada con Spring Boot y Thymeleaf.

Funcionalidades

  • Operaciones Básicas: Suma, Resta, Multiplicación, División
  • Operaciones Avanzadas: Factorial, Fibonacci, Raíz Cuadrada, Potencia
  • Interfaz Web: Desarrollada con Thymeleaf
  • Arquitectura: Patrón MVC con Spring Boot

Tecnologías

  • Java 17
  • Spring Boot 3.2.5
  • Maven
  • Thymeleaf

Inicio Rápido

  1. Clona el repositorio
  2. Ejecuta mvn clean install
  3. Inicia la aplicación con mvn spring-boot:run
  4. Accede a http://localhost:8080/calculator
